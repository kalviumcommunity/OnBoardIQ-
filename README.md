## Project Summary

The Employee Onboarding Insights Dashboard is a centralized platform designed to help organizations monitor and improve the onboarding experience of new employees. It integrates employee management, onboarding progress, internal tool usage, and support ticket data into a single dashboard, enabling HR teams, managers, and leadership to identify operational bottlenecks and make informed decisions
